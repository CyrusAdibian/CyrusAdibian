# Hello There, This is Cyrus 👋

My primary focus lies at the intersection of **low-level system architecture, cybersecurity, and AI-driven automation**. I am deeply invested in engineering highly resilient, multi-platform software environments from the kernel up. Specifically, my current research and development efforts revolve around:

* **Compiler Design & Execution Environments:** Architecting custom compilation pathways (to ARM Assembly and C#), building sandboxed virtual machines, and developing simplified kernel execution models for untrusted code.
* **Proactive Security & Threat Modeling:** Implementing multi-layered defense mechanisms, strictly separated encryption cores, and applying NIST-CNF frameworks to build systems that prioritize execution safety and resilience against modern vulnerabilities.
* **AI-Augmented Development:** Designing self-healing execution models where AI actively monitors, parses syntax, and provides real-time error recovery and troubleshooting directly within the compilation pipeline.
* **Deep Systems Programming:** Pushing the boundaries of performance and hardware interaction across both the Apple ecosystem and Windows environments using C, Swift, Rust, and Assembly.
